# Pizza Online Ordering App

Web Programming with Python and JavaScript(**Django Web Framework and Bootstrap4** )

## Description
A Web Application in which a *user* can register and then login to order food products. It has a `cart` feature where one can order products in **bulk**.
Uses Basic Frontend Stuff.
Uses Factory Method Creational Pattern for creating objects in the Model View Controller (MVC) Framework
Has Features like Cart , Add ,Drop , Email Confirmation after Placing Order, Register 
Application on Heroku through Heroku CLI and maintained a PostgreSQL database
